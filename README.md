# Simple routes

Your task is to finish this incomplete read only API.

The goal is to be able to fetch blog posts, products and users both with an endpoint that returns all the available data AND with an endpoint that returns individual objects by ID.

## Requirements

Add the following endpoints

- `GET /user` - returns all users
- `GET /blog` - returns all blog posts
- `GET /product` - returns all products

- `GET /user/1` - returns the user with the given ID
- `GET /blog/1` - returns the blog post with the given ID
- `GET /product/1` - returns the product with the given ID
